# Exam 3 Review Quiz

A self-contained, single-file HTML study app covering three anesthesia/CRNA exam units: **Patient Monitoring**, **Monitoring Ventilation**, and **Infection Control**. No build step, no dependencies, no internet connection required — just open the file in a browser.

## Question Bank

**450 questions total**, drawn directly from the Exam 3 source material:

| Unit | Questions | SATA (select-2) |
|---|---|---|
| Patient Monitoring | 100 | 3 |
| Monitoring Ventilation | 100 | 8 |
| Infection Control | 50 | 7 |
| Comprehensive Review (all units) | 200 | 14 |

Every question includes a written explanation of the correct answer.

## Features

- **Unit selection** — quiz any combination of the four banks
- **Configurable length** — set a question count or use the whole pool
- **Shuffle** — randomize question order and/or answer order
- **Instant feedback** — color-coded correct/wrong with a "Why" explanation on each question (toggleable)
- **Two question formats** — standard multiple choice and SATA ("select all that apply," capped at two picks)
- **Running score** and progress bar during the quiz
- **Results screen** — score ring, percentage, and a full missed-questions review
- **Retry missed only** — re-drill just the questions you got wrong
- **In-memory only** — no browser storage, no tracking, fully offline

## Usage

Open `Exam3_Review_Quiz.html` in any modern web browser (Chrome, Firefox, Safari, Edge). That's it — pick your units, choose your options, and start.

To host it on GitHub Pages, push the HTML file to your repo and enable Pages in the repo settings; the quiz will be served at your Pages URL.
